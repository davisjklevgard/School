Problem Statement

A program is required that will receive two numbers from a user, calculate the sum, difference, product, and quotient of the numbers. The program will display the two input numbers and the calculated results with identifying text.

1. Nouns and Verbs

Identify the nouns and verbs in the problem statement. Concentrate on the important nouns and verbs that are relevant to the problem.

PROBLEM STATEMENT:

Nouns: two numbers, sum, difference, product, quotient, identifying text

Verbs: receive, calculate, display

2. Defining Diagram

Convert the nouns and verbs into variable names and processes and create a defining diagram. Follow the course naming standards for inputs and outputs.

INPUTS:
  firstNumber
  secondNumber

PROCESSING:
  read firstNumber
  read secondNumber
  create sum
  create difference
  create product
  create quotient
  
  display sum difference product quotient

OUTPUTS:
  sum difference product quotient

3. Solution Algorithm

Create a solution algorithm based on the defining diagram. (Don't forget the algorithm name and END).

SOLUTION ALGORITHM
  MathProblems
    Initialize firstNumber to 0
    Initialize secondNumber to 0
    
    Prompt for firstNumber
    Prompt for secondNumber
    
    Convert firstNumber
    Convert secondNumber
    
    sum = firstNumber + secondNumber
    difference = firstNumber - secondNumber
    product = firstNumber * secondNumber
    quotient = firstNumber / secondNumber
    
    Display "Entered numbers: " + firstNumber + " " + secondNumber +  "\nSum: \n" + sum + "\nDifference: " + difference + "\nProduct: " + product + "\nQuotient: " + quotient
  END
