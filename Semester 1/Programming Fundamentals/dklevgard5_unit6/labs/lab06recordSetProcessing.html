Lab Steps

  • Open this file in Firefox and Notepad++
  • Add your name in the appropriate spot
  • Open the file unit6/labs/js/lab06recordSetProcessing.js in Notepad++. The code is already there to read the records and display each record on it's own line
  • In that JavaScript file you'll find existing code that's looping through a set of data from a database. You'll need to modify it so it displays some totals.
  • Look for the lines with ADD CODE HERE. Add variables and code so that you add the following output after the current output.
    • The total count of all inventory items. (Use the current stock amount to determine the number of records in the current inventory record.)
    • The total number of inventory records. (Count how many records you process)
  • Don't think too hard about this! You just have to do some simple addition to make this work. The existing code is simply looping over a collection of records, and you're adding a count of the number of records you process and tallying up a running total.
