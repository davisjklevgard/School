Final Project

The Chance of Dice!

Rules of the Game

Game's Problem Statement

          Write a program that asks for the amount to bet. This will loop as long as
          the user wishes to bet. Have them type "cash out" to stop betting. The amount of 
          money a user has will appear with the prompt for the bet. A random dice roll will
          determine what happens with the amount betted. When the user is done, 
          or the money drops to zero, there will be an output of what the user betted and 
          how they did.
          
          Nouns:
          amount, money, prompt, dice roll, output
          Verbs:
          asks, stop, appear, determine<

Defining Diagram

        INPUT:
          amount
          
        PROCESSING
          prompt user for amount
          display dice roll
          display dice result
          
        OUTPUTS:
          dice roll
          money

Solution Algorithm

        bettingOnDice
          initialize money to 1000000
          initialize bet
          initialize lost to money minus bet
          initialize won to money plus bet times 2
          initialize dice
          
          WHILE bet does not equal "cash out" or money > 0 
            prompt for amount of bet
            check for errors in amount of bet
            roll the dice
            IF dice equals 2, 3, 4, 6, 8, 10, 12
              display dice was + dice roll + you lost
              display lost
              money equals lost
            ELSE IF dice equals 5, 9, 11
              display dice roll and draw
              display "Money was returned"
            ELSE IF dice equals 7
              display "The dice was 7. You won!"
              display won
              money equals won
           ENDWHILE
        END 
          
          rollDice
            return dice equals Math.random between 2-12
          END
