Problem Statement

A program is needed that computes the sum, difference, product, and quotient of two numbers. Prompt the user for both numbers and compute each result. Each result should be displayed to the user on its own line.

Create a CalculateSum function, a CalculateDifference function, a CalculateProduct function and a CalculateQuotient function. Each function should perform its own calculation, taking in the two numbers as arguments and returning the answer back to the main program. Don't forget to convert user input to numbers.

Additional Requirements

  • The user inputs must be passed to the functions as arguments.
  • The functions must return the result and must not store the result in a global variable.
  • The part of the program that calls the functions must store the result of those functions in variables.
  • Your functions must not display results to the screen.
  • Your functions must not prompt for input.
  • Your functions must not access any variables not declared inside of the function.
  • You may choose to encapsulate your main program in a main function.
